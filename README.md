# Solana Batch Transfer Pro (SOL 批量归集/分发工具)

这是一个基于 Web 前端的 Solana 链上批量转账工具，支持 SOL 原生代币及 SPL 代币的批量归集、分发和多对多转账。

## 🛡️ 安全声明

*   **纯本地运行**：本项目是一个纯静态 HTML/JS 应用，所有私钥仅在您的浏览器本地内存中处理，**绝不会**发送到任何外部服务器。
*   **开源透明**：代码完全开源，您可以随时审查源码（`index.html`）。
*   **建议**：在处理大额资产前，请务必先使用小额资金进行测试。

## 🌟 主要功能

### 1. 多种转账模式
*   **多对多 (M:N)**：多个发送方分别向多个接收方转账（行数对应）。
*   **归集模式 (M:1)**：将多个钱包的资产汇总到一个接收地址（适用于资金归集）。
*   **分发模式 (1:N)**：从一个主钱包向多个地址分发资金（适用于空投或发薪）。

### 2. 全面的代币支持
*   **SOL 转账**：支持原生 SOL 代币转账。
*   **SPL Token 转账**：支持任意 SPL 代币（只需输入合约地址）。
*   **智能余额识别**：自动识别代币精度（Decimals）和余额。
*   **ATA 自动创建**：如果接收方没有代币账户，工具会自动创建 Associated Token Account (ATA) 并支付 Rent。

### 3. 高级控制
*   **金额设置**：支持固定金额、全额（MAX）、百分比（%）或随机金额转账。
*   **RPC 自定义**：
    *   内置 Alchemy / Helius 免费节点申请链接。
    *   支持自定义 RPC URL，避免公共节点拥堵。
*   **智能防限流 (Anti-403)**：
    *   内置指数退避重试机制，自动处理 RPC 403/429 限流错误。
    *   交易发送间隔随机化，降低被节点屏蔽的风险。
    *   Blockhash 缓存与过期重试逻辑，提高上链成功率。

## 🚀 快速开始

### 方式一：直接运行（最简单）
直接用浏览器打开 `index.html` 文件即可使用。

### 方式二：本地服务器（推荐）
如果您安装了 Node.js，可以使用 `http-server` 启动，体验更佳：

```bash
# 安装 http-server
npm install -g http-server

# 在当前目录启动
npx http-server .
```

然后访问 `http://localhost:8080`。

## ⚙️ 使用说明

1.  **配置网络**：
    *   在顶部选择网络（推荐使用自定义 RPC 以获得更好体验）。
    *   输入 RPC 节点地址（默认已配置 Alchemy 免费节点）。
2.  **选择代币**：
    *   转账 SOL：留空“代币合约地址”。
    *   转账 Token：输入 SPL 代币的 Contract Address。
3.  **选择模式并导入**：
    *   点击对应选项卡（M:N / M:1 / 1:N）。
    *   输入私钥（支持 Base58, Hex, JSON Array 格式）。
    *   输入接收地址。
    *   点击“确认导入”查看余额。
4.  **设置参数与执行**：
    *   设置转账金额（固定/随机/全额）。
    *   设置随机延迟（秒），避免并发过高。
    *   点击“开始执行”。

## ⚠️ 常见问题

*   **Error 403 / 429**: RPC 节点请求过于频繁。工具会自动暂停重试，您也可以尝试增加“随机延迟”时间或更换 RPC 节点。
*   **Blockhash Expired**: 网络拥堵导致交易未在规定时间内打包。工具会自动检测链上状态，如果确实失败会自动重试。
*   **Insufficient Funds**: 余额不足。请注意转账 SOL 时需要预留少量 SOL 作为 Gas 费（通常保留 0.002 SOL 以上）。
